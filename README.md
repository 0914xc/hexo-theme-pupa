## How to use it?

install **pupa** to your blog.
```bash
git clone git@github.com:0914xc/hexo-theme-pupa.git themes/pupa
```

install some plugins
```bash
npm install hexo-wordcount
```

modify the hexo configuration of `_config.yml`
```yaml
theme: pupa
```
